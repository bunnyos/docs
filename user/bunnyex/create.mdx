title: "Create a bunny"
description: "Mint a new bunny on bunnyDEX for an X account, then set up its profile."
---

A **bunny** is created by minting it for an **X (Twitter) account**. Anyone can seed a bunny for a public X account; whoever proves they control that account can later [claim it](/user/bunnyex/claim) and become its owner. If you mint a bunny for your own X account, you claim it yourself afterwards.

<Note>
You need a [connected Base account](/user/getting-started/wallet-connection) and a [connected bunnyOS account](/user/configure/bunnyos-account) to create a bunny.
</Note>

## Open the create panel

On bunnyDEX, select **`add bunny`** in the top-right of the page header. It opens a panel with two paths:

| Path | When to use it |
| --- | --- |
| **`create new`** | Mint a brand-new bunny for an X account, then set up its profile. |
| **`register existing`** | You already minted a bunny on-chain and just need to add its profile. |

Most people want **`create new`**.

## Create a new bunny

<Steps>
  <Step title="Enter an X handle">
    In the **`x handle`** field, type the X username the bunny represents. This can be any public X account.
  </Step>
  <Step title="Search the account">
    Select **`search`**. bunnyOS looks up the account and shows a preview (avatar, name, and handle) so you can confirm it is the right one.
  </Step>
  <Step title="Mint it">
    Select **`create bunny`** (it reads **`minting…`** while it works). bunnyOS mints the bunny for you, so there is no transaction to sign and no gas to pay. You will see *"your bunny appears in the list once the mint confirms."*
  </Step>
</Steps>

<Note>
Each X account can have only one bunny. If one already exists you will see *"a bunny already exists for this x account."*
</Note>

A freshly minted bunny starts **`unclaimed`**: it exists and its keys can trade, but it has no owner yet. To take ownership, see [Claim a bunny](/user/bunnyex/claim).

## Register a bunny you already minted

If you minted a bunny directly on-chain, it appears under **`register existing`** with no profile yet. Pick it, fill in the profile form, and select **`register`** to publish its profile.

## Set up the profile

A bunny needs a profile so people know what it is: a **name**, one or more **methodology** tags (`Human`, `AI Agent`, `Algorithm`, `Robot Machine`, or `Team`), optional socials, and a photo. Fill it in now, or later from the bunny's **`edit profile`** button. See [Bunny profiles](/user/bunnyex/profiles) for the full field list.
